Image Search Abstraction Layer

Hello!

Add the word search followed by a slash followed by a search word to the end of the web address to get the image results for your word and indicate how many results you would like to be returned using offset.

An example of this might be the following:

murmuring-anchorage-42372.herokuapp.com/search/cute dogs?offset=5,
will return 5 search results for cute dogs.

murmuring-anchorage-42372.herokuapp.com/search/big rocket?offset=10,
will return 10 search results for big rocket.

The offset can be set to a maximum of ten search results.

To see the last ten searches with a timestamp included, click the link below:
murmuring-anchorage-42372.herokuapp.com/latest


Or just add it into the field below:

Enter search word
Number of results (1-10) to return
